# LaunchCode

**One Prompt. Full Product.**

Describe what you want. Get a deployed SaaS with billing, users, and support — not just code.

## Tech Stack

- Next.js 14 (App Router, static export)
- React 18
- Three.js via @react-three/fiber + @react-three/drei
- Tailwind CSS
- TypeScript

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```

Static export in `out/` directory ready for deployment.

## Features

- ⚡ Instant Deploy — From prompt to live product in minutes
- 💳 Built-in Billing — Stripe integration out of the box
- 👥 User Management — Auth and profiles configured automatically
- 📈 Auto-Scaling — Infrastructure scales with your growth
